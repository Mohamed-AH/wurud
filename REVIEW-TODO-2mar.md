## Key Patterns Identified

### Issue Type A: Arabic Text in EN Fields
**Affected:** Sheikh Name (EN), Series Title (EN), Lecture Title (EN)
**Solution:** Translate to English with transliterated proper nouns

### Issue Type B: Arabic Characters in EN Slugs
**Affected:** All Slug (EN) fields (Sheikh, Series, Lectures)
**Solution:** Transliterate Arabic to Latin characters
**Example:**
- `تأسيس-الأحكام` → `tasees-al-ahkam`
- `الأول-من-الطهارة` → `al-awwal-min-at-taharah`

### Issue Type C: Stub/Incomplete EN Descriptions
**Affected:** Series Description (EN), Lecture Description (EN)
**Pattern:** `"Explanation of: [Arabic Name]"` or `"From book: [Arabic Name]"`
**Solution:** Expand to full English description matching AR description

## Translation/Transliteration Guidelines

### Common Terms
| Arabic | Transliteration | English |
|--------|----------------|---------|
| الشيخ | Al-Sheikh | Sheikh |
| شرح | Sharh | Explanation of |
| كتاب | Kitab | Book |
| درس | Dars | Lesson |
| الطهارة | At-Taharah | Purification |
| الفقه | Al-Fiqh | Jurisprudence |
| العقيدة | Al-Aqeedah | Creed/Belief |
| رمضان | Ramadan | Ramadan |
| أرشيف | Arshif | Archive |
| عن بعد | An Bu'd | Remote/Online |

### Slug Rules
1. Use lowercase Latin characters only
2. Replace spaces with hyphens
3. Remove diacritics
4. Keep slugs concise but identifiable
5. Maintain consistency across related records
